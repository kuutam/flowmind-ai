# FlowMind AI

Modern productivity SaaS dashboard built with Next.js, React, Tailwind CSS, and Framer Motion.

## Features
- Kanban task management
- Analytics dashboard
- AI productivity assistant
- Responsive SaaS UI
- Modern animations
- Dark/light mode

## Stack
- Next.js
- React
- TypeScript
- Tailwind CSS
- Framer Motion
- Recharts
- dnd-kit
